# Mac下命令行配置环境变量

1. 打开.bash_profile文件

启动终端，输入

sudo vim ~/.bash_profile

2. 编辑.bash_profile环境文件

按下i键进入编辑模式，如下图所示

编辑完成后，点esc 退出编辑模式

最后按下:wq是保存并退出

3. 更新环境变量

在终端输入如下命令

source .bash_profile

**注意：**macos 环境变量一般我们都保存到 ~/.bash_profile 中， 并且使用 source ~/.bash_profile 令它生效， 但重启后环境变量就失效了，我们还需要在用户的根目录下创建.zshrc文件。

# 解决环境变量失效问题

  1. 看当前系统使用的 shell（macos 自带 zsh）

    可以在终端输入下面命令查看当前使用的 shell

    echo $SHELL

    结果会输出 /bin/zsh， 说明当前使用的是 zsh

    查看 zsh 版本

    zsh --version

  2. 在 zsh 中配置 .bash_profile

    编辑 .zshrc 文件

    vim ~/.zshrc

    在末尾加入应用 .bash_profile 配置

    source ~/.bash_profile

    最后按下:wq是保存并退出。

  3. 重启试试
