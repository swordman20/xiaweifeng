# Node多版本管理

# 方案一：Node.js多版本管理的可视化工具

对于Mac系统，更建议采用方案二，简单

首先，我们在 Github 中找到 nvm-desktop 下载地址

nvm-desktopopen in new window

然后找到对应系统版本，选择文件进行下载安装

在安装前，建议卸载之前已经安装的node版本

下载完成后进行安装。

  • 安装打开可能需要配置一下环境变量。

最后运行成功后的页面如下所示，这个效果就和我们命令行的 nvm list 类似，会展示所有可安装的 node 版本信息

在安装 Node 前，也可以设置一下node 的镜像地址，从而加速 node 的访问

https://npm.taobao.org/mirrors/node/

小伙伴根据自己项目的需要，使用对应版本即可，然后选择应用，即可，然后命令行就可以查看对应的node版。

# 方案二：node多版本管理mac篇

适用于Mac和Linux系统

先介绍下需要安装的文件 Node版本open in new window管理工具n，可直接安装。 1: 查看node版本open in new window

node --version

2: 清除node缓存

sudo npm cache clean -f

3: 全局安装node版本管理工具n

sudo npm install n -g

4: 使用版本管理工具安装指定node版本或者升级到最新node版本

sudo n stable (安装node稳定版本)
sudo n latest (安装node最新版本)
sudo n 16.0.0 (安装node指定版本16.0.0)

5: 使用n切换node版本(mac系统不使用sudo不生效)

sudo n (使用上下箭头切换指定版本,再查看node版本看是否切换成功)

6: 删除已经安装的版本

sudo n rm 14.18.0
